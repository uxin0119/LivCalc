# System Monitor Dungeon - 게임 디자인 문서

## 📋 개요

**장르**: 전술 방치형 RPG / 인벤토리 관리 시뮬레이션
**컨셉**: 시스템 모니터링 툴로 위장한 던전 탐험 게임
**핵심 테마**: Escape from Tarkov식 하이 리스크/하이 리턴 메커니즘

---

## 🎯 핵심 게임 루프

```
1. 출정 준비 → 2. 장비 세팅 → 3. 원정 시작 → 4. 결과 대기 → 5. 보상 획득/손실 → 반복
```

---

## 🎮 주요 메커니즘

### 1. 출정 시스템 (Raid System)

#### 원정 파라미터
- **소요 시간**: 20~40분 (무작위 또는 던전 선택에 따라)
- **실시간 진행**: 원정이 시작되면 실시간으로 시간이 흘러감
- **위험도**: 시간이 길어질수록 보상 증가, 위험도 증가

#### 원정 중 상태 모니터링
전사의 상태를 시스템 모니터 UI로 표시:
- **CPU 사용률** → 체력 (HP)
- **메모리 사용량** → 스태미나
- **디스크 I/O** → 인벤토리 용량
- **네트워크 활동** → 전투/탐색 활동
- **프로세스 목록** → 현재 진행 중인 행동

### 2. 장비 세팅 (Loadout System)

#### 출정 전 준비 가능 항목
```
무기 슬롯:
├─ 주무기 (Primary Weapon)
├─ 보조무기 (Secondary Weapon)
└─ 근접무기 (Melee Weapon)

방어구 슬롯:
├─ 헬멧 (Helmet)
├─ 갑옷 (Body Armor)
└─ 배낭 (Backpack) - 인벤토리 용량 증가

소모품 슬롯:
├─ 의료 키트 (Medkit) x3
├─ 식량 (Food) x5
└─ 탄약 (Ammo) - 무기별 상이
```

#### 장비 손실 메커니즘
- **사망 시**: 모든 착용 장비 영구 손실
- **탈출 실패 시**: 50% 확률로 장비 손실
- **성공 귀환 시**: 모든 장비 + 획득 아이템 보존

### 3. 위험 관리 시스템

#### 조기 귀환 트리거
전사는 다음 조건에서 자동으로 귀환 시도:

1. **보급품 부족**
   - 식량 0개 이하: 스태미나 -50%
   - 의료 키트 0개 + HP 30% 이하: 조기 귀환 시작

2. **부상 상태**
   - HP 20% 이하: "Critical" 상태, 귀환 시도
   - 출혈 상태: 1분당 5% HP 감소

3. **인벤토리 포화**
   - 배낭 용량 100%: 더 이상 아이템 획득 불가
   - 선택: 계속 탐험 or 즉시 귀환

#### 귀환 메커니즘
- **정상 귀환**: 던전 입구까지 안전하게 이동 (5분 소요)
- **긴급 귀환**: 현재 위치에서 즉시 탈출 시도 (성공률 70%)
- **사망**: 모든 것을 잃고 빈손으로 귀환

## 💰 보상 시스템

### 시간당 보상 배율
```
20분 원정: 기본 보상 x1.0
25분 원정: 기본 보상 x1.3
30분 원정: 기본 보상 x1.7
35분 원정: 기본 보상 x2.2
40분 원정: 기본 보상 x3.0
```

### 아이템 등급
- **Common (회색)**: 기본 소모품, 낮은 가치
- **Uncommon (녹색)**: 개선된 장비
- **Rare (파란색)**: 희귀 무기/방어구
- **Epic (보라색)**: 강력한 장비, 높은 능력치
- **Legendary (황금색)**: 최고급 아이템, 특수 효과

### 루팅 메커니즘
- 원정 중 무작위로 전투/탐색 이벤트 발생
- 이벤트 성공 시 아이템 획득
- 인벤토리 공간이 있어야 습득 가능
- 귀환 성공 시에만 인벤토리에 영구 보관

## 🎲 게임 진행 흐름

### Phase 1: 준비 단계
1. 보유 장비 확인 (인벤토리)
2. 원정 난이도 선택 (예상 시간 20~40분)
3. 전사 장비 세팅
4. 소모품 배정
5. 출정 버튼 클릭

### Phase 2: 원정 진행
1. 실시간으로 시간 경과
2. 이벤트 로그 업데이트 (전투, 루팅, 이동)
3. 리소스 소모 (식량, 의료품, 탄약)
4. 조건 충족 시 조기 귀환 or 계속 진행

### Phase 3: 귀환 & 보상
- **성공**: 모든 획득 아이템 + 착용 장비 보존
- **부상 귀환**: 일부 장비 손상, 획득 아이템 보존
- **사망**: 모든 장비 상실, 빈손 귀환

## 🔄 메타 진행 시스템

### 영구 진행 요소
- **스태시 (창고)**: 획득한 아이템 영구 보관
- **전사 레벨**: 원정 성공 시 경험치 획득
  - 레벨업 시 능력치 증가 (HP, 스태미나, 운반 용량)
- **던전 해금**: 특정 조건 달성 시 새로운 던전 개방

### 경제 시스템
- **상점**: 아이템 구매/판매
- **제작**: 재료 조합으로 새 장비 제작
- **수리**: 손상된 장비 수리 (비용 소모)
