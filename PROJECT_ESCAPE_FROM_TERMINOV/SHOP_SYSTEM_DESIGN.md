# 상점 시스템 설계

## 📋 개요

전사를 원정에 보내기 위한 장비와 소모품을 구매하고, 원정에서 획득한 아이템을 판매하여 자금을 확보하는 경제 시스템.

---

## 💰 화폐 시스템

### 기본 화폐: Credits (₵)
- 아이템 판매로 획득
- 상점에서 물품 구매에 사용
- 장비 수리 비용 지불

### 시작 자금
- 신규 플레이어: 5,000₵
- 기본 장비 세트 제공 (Pistol, Basic Vest, Small Backpack)

## 💵 가격 정책

### 구매/판매 비율
- **구매 가격**: 기본 가격 (100%)
- **판매 가격**: 기본 가격의 60%
- **손상된 장비 판매**: 내구도에 비례하여 가격 하락
  - 100% 내구도: 60% 가격
  - 50% 내구도: 30% 가격
  - 10% 내구도: 10% 가격

### 수리 비용
- 손상된 장비는 수리 가능
- 수리 비용 = (최대 내구도 - 현재 내구도) × 아이템 가격 × 0.3

예시:
```
AK-47 (3,200₵ 가격, 현재 50% 내구도)
수리 비용 = (100% - 50%) × 3,200₵ × 0.3 = 480₵
```

## 🔄 상점 갱신 시스템

### 재고 관리
- **무제한 재고**: 기본 아이템 (Common 등급)
- **제한 재고**: 고급 아이템 (Rare 이상)
  - 매일 00:00 재고 갱신
  - 갱신 시 무작위로 3~5개의 고급 아이템 입고

### 특별 거래 (Special Deals)
- 주간 할인: 특정 아이템 20~40% 할인
- 대량 구매 할인: 소모품 10개 이상 구매 시 15% 할인

## 📊 경제 밸런스

### 수익성 가이드라인

#### 20분 원정 (성공 시 평균 수익)
- 획득 아이템 가치: 1,500₵ ~ 3,000₵
- 소모품 비용: -300₵
- 순수익: 1,200₵ ~ 2,700₵

#### 30분 원정 (성공 시 평균 수익)
- 획득 아이템 가치: 3,500₵ ~ 6,000₵
- 소모품 비용: -500₵
- 순수익: 3,000₵ ~ 5,500₵

#### 40분 원정 (성공 시 평균 수익)
- 획득 아이템 가치: 7,000₵ ~ 12,000₵
- 소모품 비용: -800₵
- 장비 손상 리스크: 높음
- 순수익: 6,200₵ ~ 11,200₵

### 파산 방지 메커니즘
- 소지금 500₵ 미만 시 "긴급 지원금" 1,000₵ 지급 (1일 1회)
- 최소한의 장비로 원정 가능하도록 보장

## 🎯 상점 관련 게임 플레이

### 전략적 선택
1. **안전한 플레이**: 저렴한 장비로 짧은 원정 반복
2. **공격적 플레이**: 고급 장비로 긴 원정 도전
3. **투자 전략**: 고급 장비 구매 후 여러 번 재사용

### 아이템 회전율
- 소모품: 매 원정마다 소비
- 무기/방어구: 평균 5~10회 원정 후 교체 필요
- 희귀 아이템: 획득 후 판매 vs 사용 선택

## 📈 향후 확장 가능성

1. **명성 시스템**: 원정 성공으로 명성 획득 → 상점 할인
2. **블랙마켓**: 불법 고급 아이템 거래 (높은 가격)
3. **경매 시스템**: 희귀 아이템 입찰
4. **거래소**: 플레이어 간 아이템 거래 (멀티플레이어 시)
